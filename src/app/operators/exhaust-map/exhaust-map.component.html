exhaustMap operator

Σημειώσεις

Πατάμε σε ένα κουμπί για να δώσουμε μια παραγγελία. Καλείται ένα service που φτιάχνει τα σουβλάκια για την παραγγελία. Αυτό το service επιστρέφει ένα observable σε 3 δευτερόλεπτα. Δηλαδή κάθε παραγγελία χρειάζεται 3 δευτερόλεπτα για να εκτελεστεί.

Αφού δώσουμε μια παραγγελία, δεν μπορούμε να πατήσουμε αμέσως ένα άλλο κουμπί για να δώσουμε μια δεύτερη παραγγελία. Πρέπει να περιμένουμε να τελειώσει η παρραγγελία για να δώσουμε μια δεύτερη παραγγελία. Οι παραγγελίες που δίνονται νωρίτερα αγνοούνται.

Το service που φτιάχνει τα σουβλάκια, αφού φτιάξει τα σουβλάκια για την παραγγελία completes. Έτσι πρέπει γιατί το exhaustMap περιμένει από αυτό το stream να γίνει completed.
